## Deprecations

- generic: `generic.GetMethodInfo` is deprecated; service info construction is now internal via `generic.ServiceInfoWithGeneric` (#1666)

## Features

- grpc: add `closeStreamTask` monitor for cleaning up cancelled gRPC client streams using `utils.SyncSharedTicker` (#1650)
- transmeta: set `ConnResetTag` on response header (`HeaderConnectionReadyToReset`) to signal clients not to reuse bad connections (#1653)
- endpoint: `endpoint.MiddlewareBuilder` may now return nil to skip adding a middleware (#1661)
- generic: add `thrift.WithIDLServiceName(name)` parse option and `generic.WithServiceName(name)` provider option to specify a target IDL service to parse (#1673)
- tool: infer `-module` from `go.mod` when not explicitly set (#1674)

## Fixes

- server: fix missing `CtxEventBusKey`/`CtxEventQueueKey` during service inline server initialization (#1659)
- server: report failure stats after panic during request processing (#1655)
- server: recover panics in `svrTransHandler.Read` and record panic error in RPC stats (#1655)
- generic: fix codec not being updated on IDL reload (#1666)
- grpc: suppress actively-triggered connection close errors during graceful shutdown using `connectionErrorfWithIgnorable` (#1677)
- tool: fix `-use` with `-tpl multiple_services` to reference each `ServiceInfo` directly (#1688)
- tool: miscellaneous kitex tool fixes for streamx codegen (#1672, #1692)
- tool: remove redundant dependency for streamx generation code (#1675)
- kitexutil: Fix nil panic in `GetCallerAddr`/`GetCallerIP` by checking `Address()` for nil (#1690)
- server: Fix log recover to use raw recover value instead of potentially nil error (#1693)

## Improvements

- codec/thrift: Add `netpollSkipDecoder` optimized for netpoll `Reader`, bypassing the generic `thrift.SkipDecoder` path; exposed via `netpollByteBuffer.NetpollReader()` (#1643)
- remote: improve server graceful shutdown to avoid EOF when idle connections receive new requests by marking `inGracefulShutdown` and setting `ConnResetTag` on responses (#1681)
- generic: enable `WriteHttpValueFallback` and `ReadHttpValueFallback` in `DefaultHTTPDynamicGoConvOpts` (#1667)
- tool: replace fastcodec with frugal in codegen when annotated (#1696)
- generic: Remove architecture-specific build constraints for dynamicgo, enabling on all platforms (#1701)
