## Breaking Changes

- codec/thrift: Remove `MessageReader`, `MessageWriter` interfaces, `BinaryProtocol` type alias, `NewBinaryProtocol`, `UnmarshalThriftException` (all previously deprecated) (#1615)
  Migration: use `github.com/cloudwego/gopkg/protocol/thrift`.
- generic: Remove `generic.ServiceInfo(pcType)` (deprecated since v0.11.0) (#1616)
  Migration: use `generic.ServiceInfoWithGeneric(g)` instead.
- generic: Remove `generic.DeprecatedGenericServiceInfoAPIKey` constant (#1616)
- generic: `generic.Args` and `generic.Result` are now type aliases (#1616)
- generic: Change `descriptor.FromThriftTType(t thrift.TType) Type` -> `FromThriftTType(v interface{}) Type` (#1555)
  Migration: use `descriptor.Type(t)` instead.
- deps: Remove `github.com/apache/thrift` from go.mod (#1615)
  Migration: add as direct dependency if your code depends on it transitively.

## Deprecations

- codec/thrift: `ThriftMsgFastCodec` type alias is deprecated; use `github.com/cloudwego/gopkg/protocol/thrift.FastCodec` directly.

## Features

- transport: Add `transport.STREAMING` and `transport.TTHeaderStreaming` protocol constants (#1541)
- streamx: New TTHeader streaming support (`streamv2`) with new packages `pkg/streamx/`, `client/streamxclient/`, `server/streamxserver/`. Includes `streamx.GenericClientStream`, `streamx.GenericServerStream`, stream middleware interfaces (`StreamMiddleware`, `StreamRecvMiddleware`, `StreamSendMiddleware`), and client options `streamxclient.WithStreamRecvTimeout`, `streamxclient.WithStreamRecvMiddleware`, `streamxclient.WithStreamSendMiddleware`. (#1541)
- generic: Support thrift streaming for JSON generic client (#1467)
- generic: Expose generic streaming over gRPC (#1627)
- generic: Support `WithParseMode` option to set ParseMode per client via `ThriftIDLProviderOption` (#1536)
- generic: Support `GoTagOption` to remove go.tag annotation (#1526)
- generic: `generic.ExtraProvider` interface and `generic.CombineServiceKey` constant for providing extra info from generic implementations (#1616)
- grpc: Support gRPC graceful shutdown -- server sends GOAWAY and drains in-flight streams (#1556)
- tool: Support generating frugal codec for specific structs (`-frugal-struct` flag) (#1607)
- tool: Call `FastWriteNocopy` in generated `FastWrite` method (#1599)
- tool: Support non-recursive code generation for incremental updates (#1630)
- tool: Use embedded thriftgo as default option (no longer requires external thriftgo binary) (#1606)
