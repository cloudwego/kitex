## Breaking Changes

- remote: Change `ByteBuffer.ReadBinary(n int) ([]byte, error)` -> `ReadBinary(p []byte) (int, error)` (#1512)
  Migration: pre-allocate slice and pass it in.
- remote: Rename `ByteBuffer.MallocLen()` -> `ByteBuffer.WrittenLen()` (#1512)
- codec/thrift: Remove `BinaryProtocol` type and `NewBinaryProtocol` constructor (#1512, #1441)
  Migration: use `github.com/cloudwego/gopkg/protocol/thrift` instead.
- codec/thrift: Remove `MessageWriter`, `MessageWriterWithContext`, `MessageReader`, `MessageReaderWithMethodWithContext` interfaces (#1512)
  Migration: use `github.com/cloudwego/gopkg/protocol/thrift` instead.
- codec/thrift: `thriftCodec.Marshal`/`Unmarshal` now use `bufiox.Reader`/`bufiox.Writer` internally (#1512)

## Deprecations

- bthrift: `pkg/protocol/bthrift` is deprecated. Use `github.com/cloudwego/gopkg/protocol/thrift` instead. Types like `bthrift.BinaryWriter` and unknown-field helpers are marked deprecated. (#1441)
- generic: `generic.ServiceInfo(pcType)` is deprecated in favor of `generic.ServiceInfoWithGeneric(g)` (#1408)

## Features

- retry: Add `retry.MixedPolicy` and `retry.NewMixedPolicy(delayMS)` integrating failure retry and backup request into a single policy. New type `retry.MixedType`. Build with `retry.BuildMixedPolicy(p)`. (#1509)
- codec: Add customized `codec.PayloadValidator` interface for payload checksum validation over TTHeader, with built-in CRC32 support via `codec.NewDefaultCodecWithConfig(CodecConfig{CRC32Check: true})` (#1478)
- generic: Support gRPC JSON generic call for Protobuf services on client side (`grpcjsonpb`) (#1411)
- generic: Generic fastcodec -- generic call now uses fast Thrift codec path (#1424)
- grpc: Add `nphttp2.GetTrailerMetadataFromCtx(ctx)` to retrieve gRPC trailer metadata from context (#1491)
- grpc: Server now returns cancel reason in context error (#1514)
- kitexutil: Add `kitexutil.GetCallee(ctx)` helper (#1492)
- bthrift: Add `TransportException`, `NewTransportException`, `ProtocolException`, `NewProtocolException`, `PrependError` (#1420)
- genericclient: Add `StreamingServiceInfo(g Generic)` for creating streaming service info (#1465)
- tool: Embed thriftgo into kitex tool (#1479)
- tool: Support generating multiple handlers for multiple services (#1425)
- tool: Support updating import path for PkgInfo (#1513)
